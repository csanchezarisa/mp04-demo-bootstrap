Inputs

Els imputs son elements que l'usuari ha de completar, ja sigui amb lletres o marques

Input normal

Introdueix un text per veure com s'actualitza el placeholder:
<input type='text' class='form-control' placeholder=''>

Input checkbox

Si us plau introdueixi el vostre email: 
<input type="text" class="form-control" id="email" placeholder="email">
<div class="form-check-inline">
    <label class="form-check-label">
        <input type="checkbox" class="form-check-input" value="">Vols que t'enviem informació sobre els futurs projectes de Puto Truño Studio?</input>
        <input type="checkbox" class="form-check-input" value="" disabled>No vull rebre mails</input>
    </label>
</div>
